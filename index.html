API-Einstellungen

Zulassungsbescheinigung scannen — API-Zugang

Um Zulassungsbescheinigungen automatisch zu scannen, wird die Claude API benötigt. Wählen Sie eine der folgenden Methoden:

ⓘ Den API-Key finden Sie unter console.anthropic.com → API Keys.

Wird nur lokal in diesem Browser gespeichert (localStorage). Wird nirgendwo gesendet außer an api.anthropic.com.

So starten Sie den lokalen Proxy:
  1. Node.js installieren: nodejs.org
  2. Die Datei proxy.js in denselben Ordner wie die HTML-Datei legen
  3. Terminal/Eingabeaufforderung öffnen im Ordner
  4. Eingeben: ANTHROPIC_API_KEY=sk-ant-... node proxy.js
  5. Browser-Tab neu laden

Einstellungen

Speicherordner für PDFs

Der Browser speichert PDFs immer im Standard-Download-Ordner. Tragen Sie hier Ihre gewünschten Pfade ein — diese werden als Hinweis gespeichert. Um den tatsächlichen Download-Ordner zu ändern, stellen Sie das direkt in den Browser-Einstellungen (Chrome: ⋮ → Einstellungen → Downloads) ein.

ⓘ Anleitung: Öffnen Sie Chrome → Einstellungen → Erweitert → Downloads → Speicherort ändern auf Ihren gewünschten Ordner. Deaktivieren Sie „Vor dem Herunterladen fragen" damit PDFs direkt gespeichert werden.
